# scaling-umbrella
Springboot + ReactJS + MySQL

Employee Management System

* CRUD Features
  -Create Employee
  -List Employee
  -Update Employee
  -Delete Employee

* Front-End Technology stack used:
  -React
  -ES6 Modern Javascript 
  -NodeJs and NPM
  -VS Code IDE
  -React App CLI
  -Boostrap 4.5 and Axios HTTP Library

* Server-Side technology stack used:
  -SpringBoot v2+
  -SpringData JPA(Hibernate)
  -Maven 3.2
  -JDK 1.8
  -Embedded Tomcat 8.5+
